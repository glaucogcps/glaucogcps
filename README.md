# Olá, eu sou o Glauco! 

Sou estudante de **Engenharia de Computação na Unicamp** com foco em sistemas dinâmicos, controle e processamento de sinais. Atualmente, desenvolvo pesquisa de Iniciação Científica (FAPESP) na área de **Controle Robusto**.

### 🛠️ Áreas de Atuação e Interesse
- **Controle & Automação:** Controle Robusto ($H_2/H_\infty$), Sistemas Não Lineares, Sistemas Invariantes no Tempo e Markovianos.
- **Matemática Aplicada:** Otimização (Linear e Não Linear) e Desigualdades Matriciais Lineares (LMIs).
- **Processamento de Sinais:** Análise de sinais digitais e sistemas de comunicação.
- **Ciência de Dados & IA:** Aplicações científicas em geral (física, eletrônica, química, matemática, pesquisa operacional).

### 🔬 Destaque de Pesquisa
Meu foco principal é o estudo comparativo de métodos de cômputo de custos para sistemas com **incertezas politópicas**, utilizando ferramentas avançadas em **MATLAB** para a resolução de problemas de controle via LMIs.

### 💻 Stack Técnica
- **Linguagens:** MATLAB, Python, C.
- **Ferramentas:** Simulink, Jupyter Notebooks, LaTeX, Quartus.
